# festival-site - コンテンツの編集方法

このリポジトリでは、初心者が簡単にサイトの画像や文章を編集できるように、`assets/content.yml` を使ってコンテンツを管理しています。

## 使い方（編集手順）

1. `assets/content.yml` をテキストエディタで開きます。
2. `hero` の設定でヒーロー領域のタイトル・日時・ボタンを編集します。`date` を変更するとカウントダウンがそれに合わせて動きます。
3. `cards` 配列に項目を追加（`type: iframe` / 省略時はテキストカード）。
   - 画像を追加する場合は `image: assets/your-image.jpg` のように指定してください（`assets/` に画像を置くのが簡単です）。
   - ファイルをダウンロードさせたい場合は `download: true` を追加します。

例（抜粋）:

```yaml
- id: poster
  title: "ポスター"
  text: "今年の公式ポスターです。"
  image: "assets/sample-poster.jpg"
  image_alt: "公式ポスター"
```

## ローカルで確認する方法
ローカルファイル（file://）では一部ブラウザが外部リソースや iframe の読み込みを制限することがあります。簡易サーバを立てて確認することをお勧めします。

PowerShell の例（ワークスペースのルートで実行）:

```powershell
py -3 -m http.server 8000
```

ブラウザで http://localhost:8000 を開くとプレビューできます。

## 参考
- YAML の書式はスペースでインデントしてください（タブは避ける）
- js-yaml を使ってブラウザ側で YAML をパースしています（CDN 経由）
